### ğŸŸ¢ Chrome Installation

1. **Download** [`tsugi-0.1.0-chrome.zip`](https://github.com/tsugi-sync/tsugi/releases/download/v0.1.0-alpha/tsugi-0.1.0-chrome.zip) from the release page.
2. **Extract** the ZIP to a folder on your computer. Remember where you extracted it.
3. Open **Google Chrome** and navigate to `chrome://extensions`.
4. Enable **Developer mode** by toggling the switch in the **top-right corner**.
5. Click **"Load unpacked"** in the top-left area.
6. Browse to the extracted folder and select the **root folder** (the one containing `manifest.json`).
7. Click **"Select Folder"** â€” Tsugi should now appear in your extensions list.
8. Click the puzzle piece icon ğŸ§© in Chrome's toolbar and **pin Tsugi** for quick access.

2. Open **Mozilla Firefox** and navigate to `about:debugging#/runtime/this-firefox`.
3. Click **"Load Temporary Add-onâ€¦"**.
4. Browse to the downloaded `.zip` file and select it directly (no need to extract).

## ğŸ› ï¸ Developer Setup

If you want to build Tsugi from source or contribute to development:

1. **Install Dependencies**
   ```bash
   pnpm install
   ```

2. **Setup Credentials**
   Copy `.env.example` to `.env` and add your tracker API keys.
   ```bash
   cp .env.example .env
   ```

3. **Run Development Server**
   ```bash
   pnpm run dev           # Chrome
   pnpm run dev:firefox   # Firefox
   ```

4. **Build for Production**
   ```bash
   pnpm run build           # Chrome
   pnpm run build:firefox   # Firefox
   ```

5. **Create ZIP Package**
   ```bash
   pnpm run zip             # Chrome
   pnpm run zip:firefox     # Firefox
   ```
