Features

Built to stay out of your way.

Every feature is designed to be invisible until you need it — tracking your progress silently, syncing reliably, and never interrupting your watch or read session.

Universal Auto Detection

Tsugi silently identifies what you're watching or reading — anime, manga, movies, TV shows — across 90+ platforms. No manual input, no popups, no friction.

AnimeMangaMoviesTV

Cross-Platform Sync

One-click sync to MAL, AniList, Shikimori, and Bangumi. Forward-only logic ensures your progress never accidentally regresses. Letterboxd and Trakt coming soon.

MALAniListShikimoriBangumi

Seamless Linking

Link any detected title to its tracker entry in one click. Progress tracked before linking is preserved and synced on first link.

Status Management

Update Reading, Watching, Completed, On-Hold, or Dropped status directly from the popup — no tab switching needed.

Lightweight

Zero background polling. Tsugi only activates when you're on a supported page, keeping your browser fast and your battery happy.

Privacy First

All data stays local in your browser. Tsugi never sends your history to any third-party server — only to the trackers you explicitly connect.

Integrations

Works with your trackers.

Connect once via OAuth. Tsugi handles token refresh automatically and keeps your lists in sync — whether you're tracking anime, manga, or movies.

Installation

Up and running
in minutes.

Tsugi is not yet on extension stores. Install it from source — it only takes a few steps.

View on GitHub
01

Clone the repository

git clone https://github.com/rachfiandj07/tsugi.git
02

Install dependencies

pnpm install
03

Add API credentials

Copy .env.example to .env and fill in your tracker API keys.

04

Build & load

pnpm build

Load .output/chrome-mv3 as an unpacked extension in Chrome.
